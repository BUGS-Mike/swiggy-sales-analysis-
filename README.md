# Swiggy Sales Analysis

## Project Overview
This project focuses on analyzing Swiggy sales data to discover business insights, customer trends, and sales performance patterns.

## Features
- Sales Performance Analysis
- Customer Insights
- Order Trend Analysis
- Revenue Tracking
- Data Visualization
- KPI Reporting
- Business Insights Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Excel
- Power BI
- Jupyter Notebook

## Project Type
Data Analytics Project

## Objectives
- Analyze food delivery sales data
- Identify top-performing categories
- Understand customer ordering behavior
- Generate actionable business insights
